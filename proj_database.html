Communicating with a database

This section is about sending queries to a remote database and showing results. The database is hosted on Heroku platform and contains the table "point coordinates" of points with names and x-, y- coordinates in an abstract system and the table "lines" with named lines defined by beginning and ending points. Primary key of the table "point coordinates" is column "point_id". In the table "lines" column "line_id" is primary key and columns "beg_point" and "end_point" are foreign_key which refers to primary key of table "point coordinates".

User can interact with this database by sending SQL-queries, which can be done using a web page in javascript. The web page is located HERE. It uses AJAX to send asynchronous requests to backend which resends queries to the database and it uses React to show/hide the text output of the query. After the page is loaded, current state of the table is shown. In order to send a query user should write it into a text input and push the "Send query" button.
